# Usage Instructions

*This file provides a standardized template for outlining how a specific calculator should be used. Include this in the calculator's dedicated feature directory if the calculator involves complex steps, heavy assumptions, or risk of misuse. Skip for trivial tools.*

## Overview
**Calculator:** Meat Processing Cost Calculator
**Purpose:** Estimate total processor fees for butchering, cutting, and wrapping livestock based on hanging weight.
**Target Audience:** Livestock producers, ranchers, and homesteaders selling or processing meat.

## Key Rules & Assumptions
- **Rule 1:** Processing fees are almost universally calculated based on *hanging weight*, not life weight. Hanging weight is generally about 60% of live weight for beef, and 70% for hogs.
- **Rule 2:** Slaughter/Kill fees are typically flat-rate per head.
- **Rule 3:** Cut and wrap fees scale directly with the hanging weight.

## Step-by-Step Guide
1. **Estimate Weight:** Input the number of animals and the estimated hanging weight. If you only know live weight, multiply it by 0.60 to get a rough hanging weight for beef.
2. **Enter Processor Fees:** Find your local butcher's cut sheet or fee schedule. Input the flat slaughter fee, and the per-pound cut and wrap fee.
3. **Include Extras:** Don't forget disposal fees (often required), transport fuel/time to the facility, and any specialty curing (smoking bacon, sausage casing, USDA stamping).
4. **Review Cost Per Pound:** The output calculates total processing cost per pound, which is vital to add to your break-even analysis when pricing retail meat.

## Advanced Notes / Edge Cases
- **Disclaimer:** Shrink loss means your final boxed/retail meat weight will be 25-40% lower than the hanging weight depending on bone-in vs boneless cuts. This calculator estimates the *cash cost paid to the butcher*, not the final yield of boxed meat.
