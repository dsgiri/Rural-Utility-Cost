# Free Resources Hub

A curated collection of practical, free resources for farmers, ranchers, and homesteaders.

## Components
- `FreeResourcesPage.tsx`: The main route and layout shell.
- `ResourceSection.tsx`: Groups resources by `category`.
- `ResourceCard.tsx`: Individual visual card for a resource.
- `resource-data.ts`: The system-of-record array containing all curated links.
- `types.ts`: Interface definitions.

## Design Philosophy
This page is explicitly designed NOT to be a massive directory. It should only hold "the best of the best" so beginners aren't paralyzed by choice. We want to highlight the USDA, Extension resources, and high-impact nonprofits (like Farm Aid or SARE).
