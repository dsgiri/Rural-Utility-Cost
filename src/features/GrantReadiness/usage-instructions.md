# Usage Instructions

*This file provides a standardized template for outlining how a specific calculator should be used. Include this in the calculator's dedicated feature directory if the calculator involves complex steps, heavy assumptions, or risk of misuse. Skip for trivial tools.*

## Overview
**Calculator:** Grant Readiness & Match Planner
**Purpose:** Calculate required matching funds and bridging cash flow for reimbursable rural grants.
**Target Audience:** Farmers, ranchers, and rural business owners applying for grants like USDA REAP or EQIP.

## Key Rules & Assumptions
- **Rule 1:** Grants usually require a percentage match from the applicant (e.g. 50% for USDA REAP).
- **Rule 2:** Most government grants are reimbursable, meaning the applicant must float the initial working capital before getting paid back.
- **Rule 3:** The "Reimbursable Cash Flow" tool assumes bridge loan interest is calculated against one full phase of working capital over an estimated 1-year period.

## Step-by-Step Guide
1. **Choose Your Scenario:** Use "Standard % Match" for simple budget calculation, or "Reimbursable Cash Flow" if you need to plan to float project costs.
2. **Enter Project Basics:** Input the total estimated project cost and the required match percentage (usually specified in the grant manual).
3. **Account for Past Spending (Tab 1):** Detail any money spent before grant approval, as these funds usually do not count toward your match.
4. **Divide into Phases (Tab 2):** For reimbursable analysis, estimate how many invoice draws you will submit. 
5. **Analyze Working Capital:** Check the warning blocks to ensure you have enough liquid cash (Working Capital Needed) before starting the project.

## Advanced Notes / Edge Cases
- **Disclaimer:** Different agencies have different rules on in-kind matching, prior costs, and draw schedules. Always consult the official Request for Applications (RFA).
